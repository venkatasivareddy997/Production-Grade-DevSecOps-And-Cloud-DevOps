# Module 11 – Infrastructure As Code (Terraform & Ansible)

Infrastructure provisioning and configuration management are core responsibilities in DevOps.

Instead of manually creating servers, networks, and environments, modern systems use Infrastructure as Code (IaC) to automate, version, and manage infrastructure reliably.

This module focuses on:

- Provisioning infrastructure using Terraform
- Managing configuration using Ansible
- Writing reusable and scalable infrastructure code
- Managing multi-environment deployments
- Automating infrastructure workflows using CI/CD
- Combining Terraform and Ansible in real-world scenarios
- Deploying cloud infrastructure (including EKS) using code

This is a hands-on, production-focused module that reflects real corporate workflows.

## What You Will Learn

- What Infrastructure as Code (IaC) is and why it matters
- Terraform fundamentals and workflow
- Writing and organizing Terraform code
- Using variables, outputs, and state
- Terraform modules and multi-environment setups
- Terraform workspaces and state isolation
- Data sources, imports, and provisioners
- Integrating Terraform with CI/CD pipelines
- Ansible basics and architecture
- Writing playbooks for configuration management
- Using Ansible roles and collections
- Automating tasks across servers
- Integrating Ansible with Jenkins
- Building end-to-end IaC pipelines using Terraform + Ansible
- Deploying Kubernetes infrastructure (EKS) using Terraform

## How to Complete This Module

- Complete all tasks listed in tasks.md
- Write and execute Terraform configurations
- Build reusable Terraform modules
- Use Ansible to configure infrastructure
- Run playbooks on multiple systems
- Integrate both tools into CI/CD pipelines
- Work on real-world projects combining Terraform and Ansible
- Capture logs, outputs, and issues
- Push your work to your repository
- Share your learning publicly

Infrastructure failures are common in real environments.  
Understanding how to debug and fix IaC issues is critical.

## Outcome of This Module

After completing this module, you will be able to:

- Provision infrastructure using Terraform
- Manage server configurations using Ansible
- Build reusable infrastructure modules
- Handle multiple environments using workspaces
- Automate infrastructure with CI/CD pipelines
- Combine Terraform and Ansible for real-world deployments
- Deploy cloud infrastructure including Kubernetes clusters
- Troubleshoot infrastructure and automation issues

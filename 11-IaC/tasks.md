# Module 11 – Tasks | Infrastructure As Code (Terraform & Ansible)

This module simulates how organizations manage infrastructure using code instead of manual processes.

You will:

- Provision infrastructure using Terraform
- Configure systems using Ansible
- Build reusable modules
- Automate deployments
- Work on real-world IaC projects

Complete tasks in sequence.
